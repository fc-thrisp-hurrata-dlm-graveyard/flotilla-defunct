Flotilla is a basic and extensible web framework for the Go language.

Installation

Flotilla currently has 2 dependencies: jingo & httprouter. After installing both, you can install with go get github.com/thrisp/flotilla.

Extensions

One of the primary goals of creating Flotilla is large pool of extensions, middleware, and packages that that promote code reuse and wasted effort recreating everything from nothing whenever you want to do something requiring a web framework in Go. Examples for creating extension can be found here, as well your imagination in reading the Flotilla source code.
